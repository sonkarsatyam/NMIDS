# 🛡 NMIDS - Network Monitoring & Intrusion Detection System

## 📌 Overview

NMIDS (Network Monitoring & Intrusion Detection System) is a desktop-based cybersecurity application developed in Python. It provides real-time network monitoring, host discovery, port scanning, packet analysis, and intrusion detection through an interactive graphical dashboard.

The project was developed as an educational cybersecurity project to demonstrate practical networking and intrusion detection concepts.

---

## 🚀 Features

* 🔍 Multi-threaded Port Scanner
* 🌐 ARP Host Discovery
* 📡 Live Packet Monitoring
* 📊 Real-time Traffic Statistics
* 📈 Live Traffic Graph
* 👥 Top Network Talkers
* ⚠ Real-time Alert System
* 🛡 SYN Flood Detection
* 🚨 Port Scan Detection
* 🔒 SMB & TELNET Detection
* 📋 Scan History
* 💻 System Information
* 📄 Export Reports (TXT, CSV, PDF)
* 🎯 Risk Score Dashboard
